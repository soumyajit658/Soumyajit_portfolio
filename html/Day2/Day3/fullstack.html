🛣️Roadmap of Fullstack💻

BASIC

  • HTML - The backbone of every webpage. It defines the structure of your website.
  • CSS - Adds colors, fonts, and layout styling to HTML elements.
  • JavaScript -Makes web pages dynamic and interactive.

Framework

  • Bootstrap-Helps create responsive websites fast and easily.
  • React JS -Used for building modern front-end interfaces using components.
  • Tailwind CSS-Makes styling easy with prebuilt utility classes.

Backend

  • Node JS -Lets you run JavaScript on the server.
  • MongoDB-NoSQL database for flexible data storage.
  • Express JS- Simplifies building server-side and backend apps.

Design

  • Figma-For UI/UX design and prototyping.
  • Adobe XD- Used to design and prototype app interfaces.
  • Photoshop -Used for editing photos and creating graphics.

Extra

  • Git-Tracks changes in your code efficiently.
  • Github -Stores and manages your code online.
  • GSAP JS - Adds smooth animations and motion effects to websites.
